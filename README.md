# 🌙 Sahur Checker

A premium iOS-style web app to check if your Sahur meal is **Gut Friendly** based on the Butyrate/Nasi Sejuk concept.

![Sahur Checker](https://img.shields.io/badge/Next.js-14+-black?style=flat&logo=next.js)
![Tailwind](https://img.shields.io/badge/Tailwind-3.4+-06B6D4?style=flat&logo=tailwind-css)
![Framer Motion](https://img.shields.io/badge/Framer-Motion-0055FF?style=flat&logo=framer)

## ✨ Features

- **Interactive Checklist** - iOS-style toggle switches for 5 gut-friendly items
- **Real-time Scoring** - Calculate your "Gut Score" (0-100%)
- **Smart Verdict** - Get personalized feedback based on your score
- **Bilingual** - Support for English and Bahasa Melayu
- **Persistent** - Your checklist state saves automatically

## 📋 Checklist Items

| Item | Weight | Description |
|------|--------|-------------|
| Nasi Sejuk / Resistant Starch | 25% | Rice cooled for 12+ hours |
| Pisang Muda / Fiber | 20% | Green bananas or plantains |
| Ghee / Healthy Fat | 20% | Grass-fed ghee for butyrate |
| Kurang Gula | 15% | Minimal added sugar |
| Air Kosong | 20% | Plain water intake |

## 🏆 Verdict System

- **80%+**: Usus Besi! 🛡️ - Your gut is iron-clad!
- **50-79%**: Usus Baik 👍 - Decent gut health
- **<50%**: Usus Tisu... 🤧 - Your gut needs love

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

localhost:3000Open [http://](http://localhost:3000) to view the app.
